# Run and Fix Tests

## Task

1. Run tests: `npm run test $ARGUMENTS`
2. If tests fail:
   - Analyze the failure
   - Read relevant source and test files
   - Fix the issue
   - Re-run to verify
3. Report results

## Output

- Number of tests run
- Pass/fail status
- If fixed: what was wrong and how it was fixed
